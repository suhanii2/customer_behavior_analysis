# customer_behavior_analysis
# 📊 Data Analytics Project

## 📌 Overview

This project focuses on end-to-end data analysis, starting from raw dataset processing to generating meaningful business insights. It includes data cleaning, exploratory data analysis (EDA), SQL-based analysis, and dashboard visualization.

The goal of this project is to demonstrate practical data analytics skills using real-world tools and workflows.

## 🔄 Project Workflow

### 1. Data Loading

* Loaded dataset using Pandas
* Checked structure and basic information

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Corrected data types

### 3. Exploratory Data Analysis (EDA)

* Analyzed trends and patterns
* Identified top-performing categories and products
* Visualized data using charts and graphs

### 4. SQL Analysis

* Imported cleaned data into PostgreSQL
* Wrote SQL queries to extract insights:

  * Top-selling products
  * Discount impact analysis
  * Category-wise performance
  * Ranking using window functions

### 5. Dashboard (Power BI)

* Built interactive dashboard
* Visualized KPIs like revenue, orders, and discounts
* Enabled filtering for better insights

### 6. Reporting & Presentation

* Created a structured report
* Designed a presentation using Gamma

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/suhanii2/customer_behavior_analysis.git
```

### 2. Install dependencies

```bash
pip install pandas numpy matplotlib seaborn sqlalchemy psycopg2
```

### 3. Run Jupyter Notebook

```bash
jupyter notebook
```

### 4. Setup Database

* Install PostgreSQL
* Create database (e.g., `customer_behavior`)
* Update credentials in code

### 5. Run SQL Queries

* Use DbVisualizer or pgAdmin

### 6. Open Dashboard

* Open `.pbix` file in Power BI
